# 🎯 リアルタイムチャットアプリ

オンラインで複数ユーザーがリアルタイムでチャットできるアプリケーションです。

## 📋 機能

- ✅ **リアルタイム通信** - Socket.IOを使用したリアルタイムメッセージ配信
- ✅ **ユーザー管理** - ニックネーム設定とオンラインユーザーリスト表示
- ✅ **メッセージ履歴** - チャット中のメッセージ履歴を表示
- ✅ **参加/退出通知** - ユーザーの参加と退出をシステムメッセージで通知
- ✅ **レスポンシブデザイン** - モバイルデバイスにも対応
- ✅ **美しいUI** - グラデーションデザインとスムーズなアニメーション

## 🛠 技術スタック

- **フロントエンド**: HTML5, CSS3, JavaScript (Socket.IO)
- **バックエンド**: Node.js, Express, Socket.IO
- **リアルタイム通信**: WebSocket

## 📦 インストール

### 必要な環境
- Node.js (v14以上)
- npm または yarn

### セットアップ手順

1. **依存パッケージをインストール**
   ```bash
   npm install
   ```

2. **サーバーを起動**
   ```bash
   npm start
   ```

3. **ブラウザで開く**
   ```
   http://localhost:3000
   ```

## 🚀 使い方

1. ブラウザでアプリにアクセス
2. ニックネームを入力して「チャットに参加」をクリック
3. メッセージを入力して「送信」ボタンをクリック（またはEnterキー）
4. 複数のウィンドウやタブを開いてチャットをテスト

## 🌐 オンラインデプロイのおすすめ方法

### 1. **Render** ⭐ (最もおすすめ)
- **特徴**: 無料枠あり、WebSocketサポート完全、デプロイが簡単
- **手順**:
  - Render.comにアカウント登録
  - GitHubリポジトリを接続
  - 新しいWebサービス作成
  - 環境変数とポート設定（PORT=3000）
  - 自動デプロイが有効化

### 2. **Heroku** (初心者向け)
- **特徴**: 無料枠なし（有料）だが非常に簡単
- **手順**:
  ```bash
  npm install -g heroku
  heroku login
  heroku create your-app-name
  git push heroku main
  ```

### 3. **Railway**
- **特徴**: 無料トライアルあり、使いやすい
- **手順**:
  - Railway.appにアカウント登録
  - GitHubリポジトリを接続
  - 自動デプロイ設定

### 4. **Replit** (最も簡単)
- **特徴**: 最も簡単、ブラウザだけで完結
- **手順**:
  - Replit.comにアカウント登録
  - New Replを作成（Node.js）
  - コードをアップロード
  - 実行ボタンをクリック

### 5. **VPS（AWS EC2, DigitalOcean等）**
- **特徴**: 完全にカスタマイズ可能、スケーラビリティ高
- **手順**:
  - サーバーにNode.jsをインストール
  - Git cloneでコードをデプロイ
  - PM2でバックグラウンド実行
  - Nginx等でリバースプロキシ設定

| **Replit** | ✅ | 最簡単 | ✅ | ⭐⭐⭐ |
